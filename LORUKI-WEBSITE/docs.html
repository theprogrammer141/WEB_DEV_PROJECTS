Introduction

Loruki is a cloud hosting platform built with the latest technology to ensure you get the best experience. With Loruki, you can easily deploy your applications and manage resources with ease.

Read our documentation to learn more about how to use Loruki.

Getting Started

Once you have signed up and logged in, you can start deploying your applications.

Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Installation

Mac (Homebrew)

$ brew install loruki-cli

NPM

$ npm install loruki-cli

Ubuntu

$ sudo-apt install loruki-cli
